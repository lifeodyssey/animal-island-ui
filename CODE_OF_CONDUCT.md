# Code of Conduct

This project follows a simple standard: be respectful, constructive, and
specific.

## Expected Behavior

- Assume good intent.
- Keep discussion focused on the code, design, docs, and user impact.
- Make disagreements actionable with examples, reproduction steps, or proposed
  alternatives.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Dismissive comments that do not help improve the project.
- Publishing private information without permission.

Maintainers may close issues, edit comments, or block participants when needed
to keep the project healthy.
