# Support

For questions about `animal-island-ui-tailwind`, use GitHub Issues in this
repository.

## Before Opening an Issue

- Check whether the issue reproduces with `import 'animal-island-ui-tailwind/style'`.
- Include the package version, React version, browser, and operating system.
- Include a small reproduction when possible.

## Upstream Project

This fork modernizes the original `guokaigdg/animal-island-ui` project. For
questions about the original Less implementation, use the upstream repository.
